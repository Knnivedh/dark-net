# Nivya Dark Net Installation Guide

## Installation Instructions

1. **Extract this ZIP file** to a temporary location
2. **Run `INSTALL.bat` as Administrator**
   - Right-click on `INSTALL.bat`
   - Select "Run as administrator"
3. **Wait for installation to complete**
4. **Launch** from your Desktop or Start Menu

## What Gets Installed

- Application installed to: `C:\Program Files\Nivya Dark Net`
- Desktop shortcut created
- Start Menu shortcut created

## System Requirements

- Windows 10/11 (64-bit)
- 2GB RAM minimum
- 1GB free disk space

## Uninstallation

To uninstall:
1. Delete the folder: `C:\Program Files\Nivya Dark Net`
2. Delete shortcuts from Desktop and Start Menu

## Features

- AI-powered chat interface
- Multiple AI providers (Cerebras, Groq, OpenAI)
- OCR text extraction from screen
- Voice input support
- Dark mode interface
- File upload support (PDF, DOCX, TXT)
